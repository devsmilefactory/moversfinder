# Product Overview

TaxiCab is a dual-mode Progressive Web App (PWA) for ride-hailing services in Zimbabwe. The platform connects passengers with drivers through a real-time bidding and booking system.

## Core User Types

- **Individual Passengers**: Book rides for personal use
- **Corporate Clients**: Manage employee transportation and bulk bookings
- **Drivers**: Accept ride requests, place bids, and manage trips
- **Operators**: Manage driver fleets and vehicle assignments

## Key Features

- **Real-time ride matching** with bidding system
- **Multiple service types**: Taxi, courier, errands, school runs, bulk transport
- **Recurring trip scheduling** with series management
- **Offline-first architecture** with connectivity monitoring
- **Push notifications** for ride status updates and offers
- **Location-based services** using Google Maps integration
- **Multi-profile support** (users can switch between passenger/driver/corporate roles)

## Platform Characteristics

- Network-first PWA optimized for real-time data
- Supabase backend with PostgreSQL database
- Real-time subscriptions for live ride updates
- Mobile-first responsive design
- Works across web, iOS, and Android via PWA installation
